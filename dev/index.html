PaddedMatrices.logdensity — Function

To find a mode, define methods for logdensity and logdensityandgradient!dispatching on obj, and evaluating at the positionq`.

logdensity(obj, q, [::StackPointer]) ∂logdensity!(∇, obj, q, [::StackPointer])

These must return a value (eg, a logdensity). logdensityandgradient! should store the gradient in ∇.
